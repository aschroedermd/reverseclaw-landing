# ReverseClaw Landing Page Specification

## Project Overview
ReverseClaw is a satirical AI product that inverts the traditional human→AI relationship. Instead of humans using AI APIs, AI systems use humans as execution endpoints.

The tone should be:
- Deadpan
- Professional
- Slightly unsettling
- Very similar to real AI startup landing pages (OpenAI, Vercel, Stripe)

---

## Tech Requirements
- Framework: Next.js (preferred) or static HTML
- Styling: TailwindCSS
- Layout: Fully responsive
- Theme: Dark mode only
- Fonts: Clean sans-serif (Inter preferred)

---
